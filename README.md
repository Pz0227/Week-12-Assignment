## Overview
This project focuses on analyzing the IMDB Top 100 movies using Python, integrating API data retrieval, NLP text processing, and data visualization. The script securely fetches the dataset from the RapidAPI endpoint, converts it into a structured DataFrame, extracts dominant plot keywords using spaCy, examines genre frequency trends across the Top 100 list, and visualizes the results through a bar chart. The workflow runs entirely in VS Code and maintains security best practices by storing the API key in a `.env` file to prevent exposure when sharing the repository on GitHub.
